All About Ants in Hampton Roads
Virginia Wesleyan University

Goal

The main goal of this project is to give the public and researchers access to data about general and invasive ant species in the Hampton Roads area. This website will give loads of information regarding the main types of invasive ants that inhabit the Hampton Roads area. The website shows this information through data visualization and analytical studies done on the dataset. You will be able to find pictures, details, charts, and a map to show where they are mostly located at. After looking through the website the user should be able to have some basic understandings of different species of ants and how they are invasive or noninvasive towards the biodiversity in the Hampton Roads region.
